.. _muterole:

Mute Role
=========

This command is a group, This means to use some commands you will need to add an extra argument

Using Default Command
---------------------
::

	-Muterole
Will return the current set mute role for your server

.. Attention:: 
	If your mute role no longer exists in discord but is still saved in the bot - It is deemed useless and is best you update it!
