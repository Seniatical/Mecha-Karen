ClearWarns
==========

Clear all warnings from a user

Parameters
----------
**Member [Required]**

The member to remove the warns from

**Reason [Optional]**

The reason for clearing the warns - Defaults set to `Wasn't Provided`

**Layout:**
::
	-Clearwarns <Member> [Reason="Wasn't Provided"]

Permissions
-----------
**Bot**
::
	Embed Links

**Author**
::
	Manage Messages

Cooldown
--------
5 Seconds - Per User
