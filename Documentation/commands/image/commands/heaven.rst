Heaven
======
He has ascended.
 
Parameters
----------
**Member [Optional]**

The member to ascend to heaven, Defaults to you.
 
**Layout:**
::
    -Heaven [member]
 
Permissions
-----------
**Bot**
::
    Embed Links
 
**Author**
::
    Send Messages / View Channel
 
Cooldown
--------
