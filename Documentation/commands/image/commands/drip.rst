Drip
====
Look at the drip.
 
Parameters
----------
**Member [Optional]**

A member in your server whos got that drip, defaults to you.
 
**Layout:**
::
    -Drip [Member]
 
Permissions
-----------
**Bot**
::
    Embed Links
 
**Author**
::
    Send Messages / View Channel
 
Cooldown
--------
10 Seconds - Per User
