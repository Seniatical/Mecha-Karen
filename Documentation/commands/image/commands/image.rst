Image
=====
Search millions of stock photos for your image!
 
Parameters
----------
**Query [Required]**

The text used to search for your image.
 
**Layout:**
::
    -Image <query>
 
Permissions
-----------
**Bot**
::
    Embed Links
 
**Author**
::
    Send Messages / View Channel
