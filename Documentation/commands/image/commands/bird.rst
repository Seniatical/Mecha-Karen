Bird
====
Place your desired image onto a bird.
 
Parameters
----------
**Argument [Optional]**

An argument which can be any of the :ref:`Accepted`.
 
**Layout:**
::
    -Bird [argument]
 
Permissions
-----------
**Bot**
::
    Embed Links
 
**Author**
::
