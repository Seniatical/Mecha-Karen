Swirl
=====
Kind of like the warp command except the effect is at the center of your image
and not the edges.
 
Parameters
----------
**Argument [Optional]**

An argument which can be any of the :ref:`Accepted`.
 
**Layout:**
::
    -Swirl [argument]
 
Permissions
-----------
**Bot**
::
    Embed Links
 
**Author**
::
    Send Messages / View Channel
 
Cooldown
--------
10 Seconds - Per User

-------
**There are currently no aliases for this command!**
