Decipher
========
Karen scrambles up a random word depending on the chosen difficulty and you have to try guess it within 30s
 
Parameters
----------
**Difficulty [Optional]**

Choose from either:

* Easy
* Medium
* Hard
* Impossible
 
**Layout:**
::
    -Decipher [difficulty]
 
Permissions
-----------
 **Bot**
::
    Send Messages
 
**Author**
::
    Send Messages
 
Cooldown
--------
10 seconds.

-------
There are no aliases for this command yet!
