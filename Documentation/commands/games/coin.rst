Coin
====
A coin will be tossed randomly by **Mecha Karen** for you as well as for her self making it competetive!
 
Parameters
----------
**Choice [Required]**

Choose between *Heads* or *Tails* and see if you win against her.
 
**Layout:**
::
    -Coin [choice]
 
Permissions
-----------
**Bot**
::
    Embed Links
 
**Author**
::
    Send Messages
 
Cooldown
--------
5 seconds
